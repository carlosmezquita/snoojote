# Database Migrations

This project uses [Drizzle ORM](https://orm.drizzle.team/) for database management.

## The `drizzle/` Folder
This folder contains the SQL migration files generated by Drizzle Kit.
- **`0000_...sql`**: The actual SQL commands to create/update your database tables.
- **`meta/`**: Internal metadata for Drizzle to track migration state.

## How to Apply Changes
After pulling new changes (like added indexes or new tables), you need to update your local database:

1. **Install dependencies** (if you haven't):
   ```bash
   bun install
   ```
2. **Push changes to the database**:
   ```bash
   bunx drizzle-kit push
   ```
   This command compares your local `schema.ts` with your `database.sqlite` and applies any missing changes.
