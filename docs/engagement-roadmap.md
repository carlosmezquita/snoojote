# Engagement Roadmap

Baseline from May 2026 exports:

- Communicators improved from 7.03% to 10.66%, still well below Discord's 30% healthy benchmark.
- New-member retention is flat at 18-21%, with the latest cohort at 18.05%.

## Activation

- Shorten onboarding to one primary action: ask a lightweight intro question that can be answered in under 30 seconds.
- Add a weekly "first message" prompt in the highest-traffic general channel.
- Pin a concise channel map so new members know where to talk, ask for help, and participate in events.
- Review low-traffic channels monthly and archive channels that split discussion without adding clear value.

## Recurring Reasons To Return

- Run one weekly scheduled community ritual: question thread, poll, watch party, game night, or Spaincraft prompt.
- Rotate the prompt format every month to prevent fatigue.
- Publish event reminders 24 hours before and 1 hour before, then follow with a recap that tags contributors.

## Recognition

- Use streak milestones and Peseta bonuses as the main daily participation loop.
- Highlight weekly top streaks, helpful members, and first-time contributors in the bot channel.
- Add small, visible rewards for low-friction actions before adding expensive long-term rewards.

## Measurement

- Track communicators, new-member retention, daily active chatters, streak activations, `/daily` claims, and ticket response time weekly.
- Compare the next Discord exports against the May 2026 baseline after two full weekly cycles.
- Treat a feature as working only if communicator percentage or retained members improve, not just if command usage rises.
